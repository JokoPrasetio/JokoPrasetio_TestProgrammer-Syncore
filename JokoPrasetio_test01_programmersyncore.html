1. Mengapa setiap bahasa pemrograman itu terdapat framework, apa kegunaan framework ?

Karena banyak nya kode yang harus dibuat dan terlalu lama dalam waktu pengerjaan sehingga dibutuhkan sebuah alat bantu atau kerangka untuk mempermudah programmer dalam pengerjaan. Kegunaan dari framework ialah mempermudah programmer dalam membuat sebuah projek sehingga tidak butuh waktu yang sangat lama dalam pengerjaanya

2. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?

if-else : pengulangan yang digunakan untuk memeriksa kondisi tersebut benar atau salah, jika benar maka program akan mengeksekusi didalam blok if dan jika salah maka program akan mengeksekusi didalam blok else.

while : pengulangan yang digunakan untuk mengeksekusi kelompok pernyataan berulang dari kondisi masih terpenuhi sampai kondisi menjadi tidak terpenuhi ketika kondisi tidak terpenuhi maka pengulangan akan berakhir.

do-while : pengulangan yang sama seperti while tetapi memiliki perbedaan, di do-while dapat mengeksekusi setidaknya satu kali walaupun kondisi awal tidak terpenuhi.

switch case : ialah membuat program memilih satu dari banyak opsi, dan akan mengeksekusi dari nilai yang sesuai dari yang ada di opsi tersebut.

for : pengulangan yang digunakan mengeksekusi kelompok pernyataan berulang dari kondisi yang masih terpenuhi.

foreach : ialah pengulangan yang digunakan kelompok pernyataan dalam sebuah koleksi data

3. Apa itu constructor, method, class, object, variabel, parameter, function, procedure ?

Constructor : fungsi khusus untuk menginisialisasi objek

Method : fungsi terkait objek atau kelas yang melakukan pengembalian nilai

Class : untuk membuat objek dengan properti dan method yang telah di definisikan

Object : Representasi dari sebuah kelas

Variabel : tempat menyimpan nilai yang dapat berubah

Parameter : nilai input untuk fungsi atau method

Function : blol kode melakukan tindakan dan menghasilkan nilai output

Procedure : blok kode untuk melakukan tindakan tanpa menghasilkan nilai output

8. Perbedaan HTTP dan HTTPS?

HTTP ialah protokol dasar yang digunakan untuk mentransfer data di internet

HTTPS ialah protokol lebih aman dari HTTP karena menggunakan enkripsi SSL.
